2.9Home

Memory Alpha is a collaborativeproject to create the most definitive, accurate, and accessibleencyclopedia and reference for everything related to Star Trek. The English-language Memory Alpha started in November 2003, and currently consists of 54,752 articles. If this is your first visit, please read an introduction to Memory Alpha.

2.9an introduction to Memory Alpha

introduction

Memory Alpha is a free Star Trek reference database written collaboratively by its readers. The site is a wiki, meaning that anyone, including you, can edit any article right now by clicking on the
“Edit”button that appears in the top right corner of every Memory Alpha article.

in the corner室内的角落
on the corner表面上的角落
at the corner室外的角落
可能站内也是室内吧

阿尔法记忆是由读者共同创作的免费ST资料库。这个网站是一个维基，每个人都可以通过点击每篇文章右上角的Edit按钮来编辑。

Browsing Memory Alpha

Memory Alpha is, essentially, an encyclopedia. You can start browsing by selecting a subject on the Main Page, and follow any link to a subsequent subject. All Memory Alpha articles are interlinked – that is, they have inline hyperlinks to other articles on related subjects. Alternatively, you can use the search box that can be foundat the top right of every page.

essentially /ɪˈsenʃəli/ adv. 本质上 形容词essential也有“本质的”意思
subsequent /ˈsʌbsɪkwənt/ adj. 随后的
alternatively /ɔːlˈtɜːnətɪvli/ adv. 要不然
at the top right这儿没有corner了

阿尔法记忆本质上是一个百科全书。你可以在主页上选择一个主题来浏览，之后通过点击一个行内超链接浏览后续的主题。每一篇文章都是互相关联的（包括这页!）。或者，你可以通过右上角的搜索栏来搜索文章。

If you’re feeling lucky, you could also try clicking the “Random page” link also found at the top of every page in the navigation menu. You could also try the “Memory Alpha” link in the menu to get to the recent changes, to check out the latest updates to our database.

navigation /ˌnævɪˈɡeɪʃn/ n. 导航 navigation menu 导航菜单

如果你觉得你现在运气爆棚，你可以试着点击顶部的导航菜单的“随机页面”链接。你也可以点菜单中的”阿尔法记忆”链接进入最近的变动页面，查看资料库最近的更新情况。

If there’s something we don’t cover, or you’re having difficulty finding what you’re after, just ask us in our referencedesk forum, or add the topic to our list of requested articles.

desk n. 组 如参考资料组 新闻组 百合组

如果你发现我们没有涵盖到的资料，或者你难以寻找要找的资料，请在参考组论坛提问，或者在请求文章区发帖。

Warning: Memory Alpha contains spoilers! For any television episode that hasaired in its nationwide time slot in the United States, that material is considered fair game for inclusion in articles. See our spoiler policy for more information.

spoiler/‘spɒɪlə/ n. 剧透

警告：阿尔法记忆包含剧透！对于任何已经在米国本土时间段播出的剧集，资料可以被文章涵盖。参阅我们的剧透政策获得更多信息。

Editing

Anyone is allowed to edit practically any page in Memory Alpha – yes, anyone and any page, even this one! Just click on the “edit this page” or “edit” link that can be found at the top of every article. You don’t have to be logged in. (See also: Why contribute?) Some pages have had to be protected against anonymous or even registered users by administrators though, due to repeated vandalism. These pages are the exception, not the rule.

practically/ˈpræktɪkli/ adv. 几乎
anonymous /əˈnɒnɪməs/ adj. 匿名的
vandalism/ˈvændəlɪzəm/ n. 对公共财产的破坏

在阿尔法记忆，每个人都可以编辑几乎任意一个文章——是的每个人，每一页，甚至这一页！只要点击每页上方的编辑按钮。你无需登录。但一些文章由于被不断的蓄意破坏不能被匿名或注册用户编辑。这些文章是例外，而不是规则。

If you want to start offeasy, you could find a page that is either incomplete or has an error, and correct that mistake by clicking on the “edit this page” or “edit” link. If you can think of a way to improve the page, go ahead and do it! Anyone else can come along later and fix it up if you make a mistake – just as you’re fixing someone else’s mistake. And if you’re worried about messing up an article, try editing using the wiki style in the sandbox first.

start off 开始着手

如果你想快速上手，你可以寻找一个不完整的或错误的文章，点击编辑按钮改正它。如果你想改善这篇文章，就大胆去做吧！即使你写的还有错误，任何在你之后来的人都会改正它，就像你改正别人的错误那样！别怕搞砸了这篇文章，建议你先使用沙盒模式来编辑！

If you see a link that is colored red, that is a link to an article that has not been created yet. Again, you can create one of these pages yourself. The red links lead directly to the page editing form, where you can write the article yourself. Type the article like you would a normal document, and then just click on the “Save” button to upload your contribution.

Type the article like you would a normal document即Type the article like you would (type)a normal document

如果你看到一个链接字体是红色的，说明那个链接的文章还没有被创建。同样的，你可以自己写一篇文章。点击红色链接可以直接进入编辑页面。放轻松！就像写一篇平常的文件来写，然后点击保存按钮上传！感谢你的贡献！

Of course, Memory Alpha does have a number of policies and guidelines to remember when editing a page. All contributions to Memory Alpha are released under the Creative Commons-Attribution-Non Commercial (CC-BY-NC) license. The CCL ensures that Memory Alpha will remain freely distributable forever – within some limits, of course. (See Copyrights for more information.)

guideline n. 准则

当然,在编辑页面时还要牢记一些政策和准则。所有对阿尔法记忆的贡献都被CC-BY-NC保护着。CCL确保了阿尔法记忆将永远保持自由开发——当然还是有一些限制的。

Philosophy

Some people feel uncomfortable with the concept of a wiki – it is too open, it is unreliable, and so forth. But believe it or not, this system does work – and we currently have 54,752 distinct articles to prove it!

forth adv. 向前
distinct n. 不同的

一些人对于wiki感到不适应——它太开放，太不可靠，又太激进了。但信不信由你，这个系统确实有效——而且我们现在有54752篇不同的文章来证明了！

In a nutshell, a wiki works because it is made as simple as possible for anyone to contribute, in a medium in which the sensible contributors outnumber and overpower the disruptive ones. Additionally, the wiki allows an easy, free exchange of information that allows false or misleadinginformation to be quickly corrected. (For more discussion on the viability of the wiki system, check out replies to common objections.)

in a nutshell简而言之

outnumberv.在数量上压倒

misleading adj. 迷惑性的
viability /ˌvaiə’biliti/ n. 可行性viable adj. 切实可行的

简而言之，这个wiki系统有效是因为它十分简单得每个人都能编辑。而且在这种媒介中，理智的贡献者在数量和力量上压倒了破坏性的贡献者。而且，wiki实现了一个简单自由的信息交换过程，任何错的或迷惑性的信息都会被快速纠正。（对于更多对wiki系统可行性的讨论，可以查看对常见异议的回复）

Memory Alpha uses the wiki format in order to get as many fans as possible involved in the project. Rather than numerous small and limited personal databases, Memory Alpha’s goal is to be the largest, most reliable Star Trek database available. We hope you’ll join us!

numerous/ˈnjuːmərəs/ adj.众多的

阿尔法记忆用wiki的格式使越来越多的ST迷加入这个项目。阿尔法记忆的目标不是成为众多又小又有限制的私人资料库。阿尔法记忆的目标是成为最大，最可靠的ST资料库。我们期待你能加入！
